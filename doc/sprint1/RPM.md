## Release Goal

By the end of this release, **LionsList** will be a fully functional marketplace platform that allows York University students to securely **register and log in with their YorkU email address**, as well as **create, manage, browse, and filter listings**. Additionally, users will be able to interact with each other through **messaging** and a **rating system**.

---

## Project Scope

| Epic | Goal | User Stories |
|------|------|-------------|
| User Registration and Authentication | Allow users to securely create, access, and log out of their accounts with their unique YorkU email. | 1, 2, 3, 4 |
| Search and Filtering | Allow users to efficiently search for desired listings. | 5, 6 |
| Marketplace Listings | Allow users to create and browse listings for academic materials. | 7, 8, 9, 10, 11 |
| Offer System | Allow buyers and sellers to negotiate prices on listings. | 12, 13, 14, 15, 16 |
| Rating System | Allow sellers and buyers to rate each other to build trust and assess reliability. | 17, 18, 19 |
| Messaging System | Allow users to communicate with buyers and sellers regarding listings. | 20, 21 |
